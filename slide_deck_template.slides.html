Goals in the English Premier League¶

by Mathias Brønd Sørensen¶

Investigation Overview¶

The English Premier League is my favorite football league (sorry Americans, it's not called soccer), but how different is the game from season to season? From the first match to the last match? How many goals can we expect in a match?

Let's have a look!

Dataset Overview¶

The dataset is a tidy subset version of the football matches, players and teams from Europe and is obtained from Kaggle.

Note that the above cells have been set as "Skip"-type slides. That means that when the notebook is rendered as http slides, those cells won't show up.

#plt.colorbar();
counts = h2d[0]

# loop through the cell counts and add text annotations for each
for i in range(counts.shape[0]):
    for j in range(counts.shape[1]):
        c = counts[i,j]
        if c >= 7: # increase visibility on darkest cells
            plt.text(bins_x[i]+0.5, bins_y[j]+0.5, int(c),
                     ha = 'center', va = 'center', color = 'white')
        elif c > 0:
            plt.text(bins_x[i]+0.5, bins_y[j]+0.5, int(c),
                     ha = 'center', va = 'center', color = 'black')
plt.xlabel('Home team goals');
plt.ylabel('Away team goals');
plt.title('Heat Map of the Relationship Between Home and Away Goals');
